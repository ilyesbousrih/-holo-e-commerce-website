HOLO

HOLO is a modern e-commerce web application designed for the sale of electronics and computer hardware. The platform offers a comprehensive online shopping experience, allowing users to explore and purchase a wide range of technology products, including PCs, laptops, computer components, gaming accessories, mice, keyboards, monitors, and more.

The project focuses on delivering a fast, intuitive, and secure shopping experience while demonstrating modern web development practices. Key features include product browsing, category filtering, shopping cart management, user authentication, order processing, and a dedicated admin dashboard that enables efficient management of products, categories, orders, and users.

Developed as a full-stack web application, HOLO showcases the implementation of a complete online electronics store with both customer-facing and administrative functionalities.
